Android studio 代理设置

前言

AS 自带的教程选项栏都比我多一个显然是用不了（国内特供版本

在模拟器里面再开一个代理软件 会让本来就高占用的内存直接红温 所以我选择用 PC 代理加虚拟机网络配置的方式实现模拟器也能访问外网

使用工具：

  1. Clash for Windows

  2. Android Studio emulator api 34 如下

Clash 设置

在 clash 界面进行如下设置

  • 开启 Service Mode (开启后有个绿色图标

  • 打开 TUN MODE

  • 打开 ALLOW LAN

  • 代理设置为全局代理 自行选择可行节点（耗费流量 谨慎开启
